# AngularJS Workshop Google got your back
## Follow along http://niklas-r.github.io/AngularJS-workshop
## What to expect? Throughout this workshop we'll be switching between presentation and writing code. At the end of this workshop you should have written a variant of the famous [Todo Manager](http://todomvc.com/) and feel confindent enough to use Angular in a real project.
## Why? AngularJS is an [MV*](https://plus.google.com/+AngularJS/posts/aZNVhj355G2) framework that is ideal for use when building client-side single-page apps. It's also great for creating encapsulated components embedded in a web page.
## When? If you're finding yourself building a SPA, AngularJS will be perfect for you. Gmail, Google Docs, Twitter, and Facebook all fit into the AngularJS sweet spot.
## Don't use Angular ...if you're doing game development, manipulating the DOM *a lot* or need high speed performance (using a framework comes at a cost).

Getting started


<!doctype html>

  <title>My Angular App</title>
</head>

  <div>
    <input type="text" ng-model="yourName" placeholder="Enter a name here">

  </div>

</html>
